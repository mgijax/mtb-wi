This table was developed for the special Cancer and Genomics issue of Genome Research (Bult et al., 1999. Web resources for basic cancer genetics research. Genome Research, in press). The types of resources we have included in this list are as follows (some resources belong in more than one category):
  • Animal Models:Distributors and Animal Production Services
  • Animal Models:Phenotypes for Genetically Defined and Genetically Engineered Animals
  • Cancer Genetics and Genomics
  • Pathology & Images
  • Reagents, Services, Laboratory Protocols
  • Toxicology
To add your favorite cancer research resource to this list send a message to the Mouse Tumor Biology database staff.

How to Use the Table of Web Resources for Basic Cancer Genetics Research

1. The information resources listed below are categorized as being Browsable, Queryable, Linking, or some combination of these. Browsable resources are those that allow users to navigate information using hyperlinked text. Queryable resources provide some mechanism for queries using keywords and/or controlled vocabularies. Linking resources provide mostly hyperlinks to other web sites and are often organized by a specific topic area.

2. If a cell in the table has a check mark in it, the resource listed has information related to the column category.

3. If a resources requires users to register or to subscribe (usually an access fee is involved) to gain access it is noted in the table.

4. Click on the hyperlinked URL to navigate to the listed resource
