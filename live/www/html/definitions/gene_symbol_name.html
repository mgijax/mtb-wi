Gene symbol, Name
The approved symbol and name for a genetic marker.

If the gene/locus is from the mouse the gene symbol will be hyperlinked to its corresponding Marker Detail page in the Mouse Genome Database (MGD).

If the gene/locus is not from mouse, the species of origin is indicated in parentheses after the Gene symbol. When possible, a hyperlink will connect the user to the MGD homology page for that gene/locus.
