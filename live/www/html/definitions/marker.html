Marker
The approved symbol and name for a genetic marker. The organism for the gene is given in parentheses.

If the gene/locus is from the mouse the gene symbol will be hyperlinked to the Mouse Genome Database (MGD) detailed record for that gene/locus.

If the gene/locus is not a mouse gene the hyperlink (if present) will connect the user to the MGD homology page for that gene/locus.
